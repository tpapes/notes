Kosaraju's Algorithm

Kosaraju's Algorithm finds the strongly connected components in a directed graph.

The Algorithm:

  1. let L be an empty stack.
  2. for each vertex u in the graph, if u is not visited, do Visit(u), where Visit(u) is described as follows:
    • mark u as visited
    • for each vertex v at an outbound edge of u, if v is not visited, do Visit(v)
    • prepend u to L
    At this point, we will have recursively prepended each vertex to L in post-order, meaning that no vertex can have an edge pointing to a vertex to its left.
  3. for each vertex u in L, if u is not connected, do Connect(u, u), where Connect(u, root) is described as follows:
    • mark u as connected
    • set the representative node of the component to root
    • for each vertex v at an inbound edge of u, if v is not connected, do Connect(v, root)
    At this point, all vertices will have a representative root, where all vertices in the same component share the same root.

This algorithm, in essence, searches for groups of vertices that are connected both forwards and backwards, and gives them the same representative. It does this in a somewhat roundabout way though.
